Wie ich meine erste Website selbst gebaut habe – und was ich dabei gelernt habe

20. April 2025

Die Idee, eine eigene Website zu erstellen, hatte ich schon etwas länger. Nicht mit einem Homepage-Baukastentool wie Squarespace oder WordPress, sondern komplett selbst – mit HTML, CSS und JavaScript. Ich durfte bereits im Rahmen meines Bachelorstudiums ein wenig Erfahrung mit HTML und CSS sammeln, konnte dieses Wissen jedoch nie wirklich in Projekten einsetzen. Da heutzutage viele Low-Code- oder No-Code-Lösungen verfügbar sind, wollte ich bewusst den „Old-School“-Weg gehen und mit reinem Code arbeiten.

Also habe ich mich hingesetzt, VS Code geöffnet und mit einer leeren index.html gestartet. Schritt für Schritt ist daraus eine eigene Website entstanden – mit Startseite, Blogbereich, Themenübersicht und einer persönlichen Seite über mich.

Unterwegs habe ich viel gelernt: wie man eine sinnvolle Ordnerstruktur anlegt, wie man mit CSS eine Seite gestaltet, wie JavaScript z. B. für einen Dark Mode sorgt – und wie wichtig es ist, auf korrekte Dateipfade zu achten. 😅 Besonders spannend war es, die Seite über GitHub Pages online zu bringen. Vorher war mir gar nicht klar, wie einfach man mit ein paar Klicks eine eigene Website hosten kann – kostenlos und öffentlich zugänglich. Früher war WordPress für mich der bekannteste Weg, doch dort fühlte ich mich schnell eingeschränkt. Umso begeisterter war ich, als ich entdeckte, dass GitHub mit seiner Pages-Funktion genau das bietet, was ich gesucht habe.

Natürlich lief nicht alles sofort glatt: CSS wurde nicht geladen, weil der Pfad falsch war. Der Dark Mode funktionierte erst, als ich den Scriptaufruf korrekt eingebunden hatte. Und bei GitHub musste ich lernen, dass die index.html im Hauptverzeichnis liegen muss, damit alles richtig angezeigt wird.

Rückblickend habe ich extrem viel gelernt, nicht nur über Code, sondern auch über das Zusammenspiel von Struktur, Logik und Darstellung im Web. Ich weiß jetzt, wie man eine Seite von Grund auf aufbaut, wie man Probleme findet und löst – und wie zufriedenstellend es ist, am Ende das Ergebnis zu sehen: eine funktionierende eigene Website.

Ich habe auf jeden Fall Lust, den Blog weiter auszubauen. In Zukunft möchte ich hier über Themen schreiben, die mich beschäftigen Technik, Kreativität, Selbstentwicklung und alles, was mich bewegt. Vielleicht kommt auch irgendwann eine eigene Domain dazu, ein paar fortgeschrittene Features und wahrscheinlich eine bessere Optimierung der Blog-Erstellprozesse. Aber jetzt freue ich mich erstmal, diesen ersten Schritt gemacht zu haben.

- Leon Oparin
